How to read this

  • Composite = 0.6 × tool score + 0.4 × memory score (the headline number).
  • Tool score (per case): matched ÷ expected tools, −0.1 per extra call, clamped 0–1. A "no-tool" case scores 1.0 only if the agent called nothing.
  • Memory score (per question): 1.0 if the expected answer appears in the reply (case-insensitive substring), else 0 — a fast local proxy for the on-chain LLM judge.
  • Score colors: 1.0 · partial · 0.
  • Inputs are fixed, but the model is stochastic — expect small run-to-run variation.
